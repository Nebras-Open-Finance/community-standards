# Onboarding to the Trust Framework

This section covers the steps required for an LFI to register within the UAE Open Finance Trust Framework and prepare their organisation for API Hub integration.

## Steps

| Step | Page |
|---|---|
| 1 | [Adding Users](./adding-users) — invite team members to your organisation's Trust Framework account |
| 2 | [Organisation Certificates](./organisation-certificates) — generate and upload your organisation's signing and transport certificates |
| 3 | [Creating an Application](./application) — register a software statement for your API Hub integration |
| 4 | [Keys and Certificates](./keys-certificates) — configure application-level keys for mTLS and message signing |
