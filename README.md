# UAE Open Finance Community Standards (experimental)

Community-led drafts, examples, and tooling experiments for the UAE Open Finance standards. This repo is **not** an official source of truth; it exists to prototype ideas and collect feedback before they move upstream.

## Quick start
- Prerequisites: Node.js 18+ and npm.
- Install deps: `npm install`
- Run docs locally: `npm run dev:docs` (opens VitePress on localhost with hot reload).
- Build static site: `npm run docs:build`
- Preview build: `npm run docs:serve`

## Contribution guide
- Please read `CONTRIBUTING.md` before submitting changes.
- Keep sensitive material out of the repo; use environment-specific secrets locally.
- Open issues for broken links, inaccuracies, or UX problems in the component demos.

## Notes
- Content, wording, and components are experimental and may change frequently.
- References to banks/LFIs are illustrative; confirm details against official Open Finance documentation before production use.
