# Adding Users

Organisation administrators can invite additional users to the Trust Framework portal, granting them access to manage certificates, applications, and directory listings.

## Roles

| Role | Capabilities |
|---|---|
| **Admin** | Full access — invite users, manage certificates, create/edit applications |
| **Developer** | Read and manage applications and certificates; cannot invite users |
| **Viewer** | Read-only access to organisation details and applications |

## How to invite a user

1. Log in to the Trust Framework portal.
2. Navigate to **Organisation Settings → Users**.
3. Click **Invite User**.
4. Enter the user's email address and select their role.
5. Click **Send Invitation** — the user will receive an email with a registration link.

::: info
Invited users must complete their own registration before they can access the portal. Invitations expire after 7 days.
:::

## Removing a user

To remove a user, navigate to **Organisation Settings → Users**, find the user in the list, and click **Revoke Access**. The user's session will be terminated immediately.
