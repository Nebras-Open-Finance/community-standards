# Trust Framework (Directory)

The **UAE Open Finance Trust Framework** is the central directory that maintains the registry of all participants in the UAE Open Finance ecosystem — Licensed Financial Institutions (LFIs), Third-Party Providers (TPPs), and the AlTareq Hub itself.

As an LFI, your organisation and its software statements must be registered and active within the Trust Framework before you can receive API calls from the Hub or authorise TPP requests on behalf of your customers.

## What the Trust Framework provides for LFIs

| Function | Description |
|---|---|
| **Organisation registry** | Your LFI's legal entity and contact details are registered here |
| **Certificate store** | Your organisation's signing and transport certificates are published here |
| **Software statements** | Each API Hub application you register is associated with a software statement |
| **TPP discovery** | LFIs can look up any registered TPP and verify their certificates against the directory |
